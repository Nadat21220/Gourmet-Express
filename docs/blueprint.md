# **App Name**: Gourmet Express

## Core Features:

- User Authentication: Secure user login and registration using Firebase Authentication with email/password support and password recovery flow.
- Menu Display and Exploration: Display the restaurant's menu with categories, product images, names, prices, and availability status, allowing users to browse and search for items.
- Customizable Order Creation: Enable users to customize orders by selecting sizes, quantities, and adding special notes, with dynamic price updates and promotion applications.
- Real-Time Order Tracking: Provide real-time updates on the order status with a countdown timer and visual indicators, notifying users when their order is ready for pickup, using Firestore real-time listeners.
- Secure Payment Processing: Process online payments securely with card payment integration (Stripe/PayPal) via Cloud Functions, ensuring secure handling of sensitive payment information.
- AI-Powered Order Assistance: AI tool that analyzes order patterns to provide intelligent recommendations, suggest complementary items, and highlight personalized promotions to enhance the user experience and streamline the ordering process.
- Admin Dashboard: Admin Dashboard to create, read, update, and delete items in the restaurant catalog and change order status with push notifications to the user.

## Style Guidelines:

- Primary color: A warm, inviting terracotta (#E07A5F) to evoke the cozy atmosphere of a gourmet cafe.
- Background color: Light beige (#F8F5F1), a soft, desaturated tint of the primary, creating a comforting and appetizing backdrop.
- Accent color: A gentle, golden yellow (#F2CC8F), analogous to the primary, used to highlight calls to action and key information.
- Headline font: 'Playfair', a modern sans-serif similar to Didot, bringing an elegant, fashionable feel to headings and short texts; body text: 'PT Sans', pairing perfectly for readability.
- Simple line icons with rounded edges, suggesting friendliness and efficiency.
- Clean and modern layout with clear sections and ample spacing for a seamless user experience.
- Subtle transitions and animations to enhance user interaction and provide feedback.
